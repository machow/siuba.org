Reshape tables üìù

import pandas as pd
from siuba import _, spread, gather

costs = pd.DataFrame({
    'id': [1,2],
    'price_x': [.1, .2],
    'price_y': [.4, .5],
    'price_z': [.7, .8]
})

costs
id price_x price_y price_z
0 1 0.1 0.4 0.7
1 2 0.2 0.5 0.8

Gather data long

# selecting each variable manually
costs >> gather('measure', 'value', _.price_x, _.price_y, _.price_z)

# selecting variables using a slice
costs >> gather('measure', 'value', _["price_x":"price_z"])

# selecting by excluding id
costs >> gather('measure', 'value', -_.id)
id measure value
0 1 price_x 0.1
1 2 price_x 0.2
... ... ... ...
4 1 price_z 0.7
5 2 price_z 0.8

6 rows √ó 3 columns

Spread data wide

(costs
    >> gather('measure', 'value', -_.id)
    >> spread('measure', 'value')
)
id price_x price_y price_z
0 1 0.1 0.4 0.7
1 2 0.2 0.5 0.8

Pivot wider and aggregate

If there would be multiple entries per cell in the spread wide data, then the spread() function raises an error.

This is shown below, where there are duplicate entries where id=1 and measure="a".

df = pd.DataFrame({
    "id": [1, 1, 2],
    "measure": ["a", "a", "b"],
    "value": [8, 9, 10]
})

df >> spread("measure", "value")
ValueError: Index contains duplicate entries, cannot reshape

Use the pandas pivot_table function to deal with this situation.

df.pivot_table(columns="measure", values="value", index="id", aggfunc=list)
measure a b
id
1 [8, 9] NaN
2 NaN [10]

Notice that the top-left entry is a list of two values, [8, 9]. The aggfunc argument is able to reduce those values down to one.

For example, by taking the mean.

df.pivot_table(columns="measure", values="value", index="id", aggfunc="mean")
measure a b
id
1 8.5 NaN
2 NaN 10.0
Note

siuba can pipe to the pandas DataFrame‚Äôs pivot_table method as follows.

(df
    >> _.pivot_table(...)
)

Where you would replace ... with your arguments. See flexible piping for more details.
