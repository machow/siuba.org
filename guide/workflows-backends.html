Demo query

We‚Äôll use the following lazy pipe to demonstrate each backend. It groups by the cyl column, then filters by mpg.

filter_mpg = (
    _
    >> group_by(_.cyl)
    >> filter(_.mpg < _.mpg.mean())
)

For example, here is the pipe called on pandas data.

cars >> filter_mpg
